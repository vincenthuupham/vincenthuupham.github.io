This project was my introduction to working with microcontrollers and basic hardware.

I used an ESP32 development board to control an LED by writing a simple program that repeatedly turns a GPIO pin on and off at fixed time intervals. This creates a blinking effect.

Through this project, I began to understand how a microcontroller interacts with physical components. The ESP32 acts as a programmable controller, where specific pins can be configured as outputs to supply voltage. By setting a pin HIGH or LOW in code, I was able to control the flow of current through the circuit.

I also learned the importance of basic circuit components. The LED requires a current-limiting resistor to prevent damage, and the circuit must form a complete path between the output pin and ground for current to flow.

On the software side, I used the Arduino IDE to write and upload code to the ESP32. This introduced me to the structure of embedded programs, including setup and loop functions, as well as basic timing control using delays.

Although this was a simple project, it established foundational concepts in embedded systems, including GPIO control, circuit design, and the interaction between software and hardware.
